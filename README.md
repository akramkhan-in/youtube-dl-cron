## Description
youtube-dl-cron is a cron script that automatically updates youtube-dl when a new version is released.

## Installation
cd youtube-dl-cron

sudo cp update-youtube-dl /etc/cron.daily/update-youtube-dl

chmod +x /etc/cron.daily/update-youtube-dl
